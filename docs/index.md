
# 简介

作为一名前端开发，「表单开发」是我们的家常便饭，一般我们需要做以下重复性工作：
* 编写模板
* 编写校验规则
* 获取数据，提交表单

综上，我们希望能减少重复性工作，通过配置自动生成表单模板。   
于是便有了[VSchemaForm](http://vueschemaform.aegis-info.com/doc/jieshao)的Vue表单解决方案。

此文档主要是具体使用例子。
